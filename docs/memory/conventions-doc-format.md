# 中文文档排版规范

> 适用于 `docs/` 目录下所有文档、README、plan 文档。

## 规则

1. **语言**：项目面向中文使用者，所有用户文档使用简体中文
2. **Markdown 格式**：段落间严格换行
3. **中英文空格**：中文与英文、中文与数字之间空一个空格
4. **标点规范**：中文上下文使用中文标点（中文引号、中文冒号），严禁英文引号或冒号；日期、时间及英文上下文遵循英语规范
5. **加粗空格**：加粗内容前后（`**` 前后）空一格，例如：`前 **加粗** 后`
6. **缩写标注**：首次出现英文缩写或专有名词时括号备注全称和中文，后续无需再标注；已是全称或过于简单的概念无需标注
